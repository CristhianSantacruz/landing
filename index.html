Spring Boot

Spring Boot es un marco de trabajo de código abierto para crear aplicaciones Java de manera rápida y con un mínimo esfuerzo. Facilita la configuración y el desarrollo de aplicaciones basadas en Spring.

Quarkus

Quarkus es un marco de trabajo de código abierto diseñado para optimizar el desarrollo de aplicaciones Java nativas en la nube. Proporciona tiempos de inicio rápidos y un bajo consumo de memoria.

Postman

Postman es una herramienta de colaboración para el desarrollo de APIs. Permite a los desarrolladores diseñar, probar y depurar APIs de manera eficiente, facilitando la creación de servicios web.
